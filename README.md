# Flow-Map-in-R
Transport de mercaderies per carretera de vehicles espanyols (tones i tones-quilòmetre). Relacions comunitat autònoma d'origen i comunitat autònoma de destinació.

Dataset -> https://apps.fomento.gob.es/BDOTLE/visorBDpop.aspx?i=281

Eina emprada per realitzar el gràfic -> https://www.r-project.org/
